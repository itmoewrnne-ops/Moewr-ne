## 🚀 Option A: Vercel (Website buuxda – 5 daqiiqo)

### 1. Taabo link-kan:
**https://vercel.com/new/clone?repository-url=https://github.com/itmoewrnne-ops/Moewr-ne**

### 2. Sign in GitHub

### 3. Deploy taabo (ma beddelo wax)

### 4. Deploy dhammaado ka dib – Environment Variables ku dar:
- **Settings** → **Environment Variables**
- `DATABASE_URL` – Neon: https://neon.tech (New Project → copy connection string)
- `JWT_SECRET` – `moewr-secret-2024-minimum-32-characters-long`

### 5. Redeploy taabo

### 6. Domain: Settings → Domains → Add `moewr-ne.so`
